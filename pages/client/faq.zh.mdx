# 常见问题

## 一般问题

### Le-AI 是什么？

Le-AI 是一个[开源的](https://github.com/LTopx/Le-AI)，基于 **OpenAI API** 的第三方客户端。其愿景是通过 OpenAI API 提供的能力，为用户提供高质量的聚合 AI 服务。包括了：

- 基础的问答功能
- 文本生成
- 图片生成
- 代码生成
- 文字转语音
- AI 角色库
- 多平台客户端
- 等等功能

### 可以免费使用 Le-AI 吗？

因为 Le-AI 的大部分功能都是由 OpenAI 驱动的，所以只需要拥有 OpenAI 的 API Key，就可以免费使用 Le-AI 的**大部分功能**。相关的费用直接通过您的 API Key 扣除。

与此同时，为了方便部分没有 OpenAI API Key 或者访问不便的朋友，我们也提供了 [**OpenAI API 代理**](/server/introduction) 服务。

### 可以使用自己的代理服务吗？

可以的。Le-AI 默认使用的是 OpenAI 官方的服务，你也可以配置自己的代理服务。除此之外，你也可以使用 [**Le-AI API**](/server/introduction) 来体验 Le-AI 的功能。

### 可以使用除 OpenAI 以外的其他模型吗？

在老版本中，Le-AI 支持 OpenAI，Azure OpenAI，OpenRouter 实现的 Claude AI 等模型。由于一些原因，暂时移除了除 OpenAI 以外的其他模型。后续会考虑将 Claude AI 重新加入。
